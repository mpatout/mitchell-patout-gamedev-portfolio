# Gameplay Evidence Index

This index tracks optional gameplay media artifacts.

## Current Submission Policy

- Gameplay clips are not required for current repository submission readiness.
- This repository intentionally excludes placeholder gameplay clips.
- If requested by a reviewer, share captured gameplay clips separately or add them in a follow-up update.
