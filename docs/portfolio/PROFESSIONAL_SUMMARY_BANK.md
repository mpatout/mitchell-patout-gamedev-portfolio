# Professional Summary Bank

Use this file as a reusable bank of concise, professional statements for resumes,
profiles, outreach messages, and recruiter conversations.

## Usage Rules

- Keep every claim truthful and evidence-backed by repository artifacts.
- Prefer concise outcomes and concrete systems over buzzwords.
- Tailor snippets to audience and character limits.
