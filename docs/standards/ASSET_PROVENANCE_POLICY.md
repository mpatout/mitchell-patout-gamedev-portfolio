# Asset Provenance Policy

All non-original assets must be explicitly documented.

## Allowed Asset Sources

- Original assets created for this repository
- CC0 assets
- Assets with explicit permissive licensing suitable for redistribution

## Required Provenance Fields

For each third-party asset, record:

- Asset name
- Source URL
- License type
- Author or publisher
- Date retrieved
- Project usage location

## Enforcement

- Every engine project must include `ASSET_PROVENANCE.md`.
- Pull requests adding assets without provenance documentation are blocked.
