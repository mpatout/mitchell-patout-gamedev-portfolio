# ASSET_PROVENANCE — Defold: Pulse Grid

## Summary

No third-party assets. All visual output is produced by Defold's built-in GUI
box/text nodes using `gui.set_color` and `gui.animate`. No sprite atlases,
textures, sound files, or font files beyond the Defold built-in default font
(`/builtins/fonts/default.font`) are referenced.
