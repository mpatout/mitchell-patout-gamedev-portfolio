# CHANGELOG - Panda3D: Lane Drift

- Playable Panda3D vertical slice (`main.py`) with lane movement, obstacle spawns,

- Portfolio support docs: acceptance test, profiling baseline, release runbook.
- Engine packaging + quality gate scripts.
