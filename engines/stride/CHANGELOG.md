# CHANGELOG - Stride: Lane Surge

## [0.1.0] - 2026-05-22

### Added

- C# gameplay simulation baseline (`Program.cs`) with lane logic, obstacle flow,
  score progression, and collision-based run termination.
- Deterministic seed mode via `STRIDE_SEED`.
- JSON run trace export and best-score persistence under `engines/stride/runtime/`.
- Acceptance test, profiling baseline template, and release runbook documents.
- Stride packaging and quality gate scripts.
